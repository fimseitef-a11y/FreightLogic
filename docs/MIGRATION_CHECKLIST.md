# Migration Checklist

This checklist is designed to verify that the reorganization of the FreightLogic repository has been completed successfully. Each step should be checked off once completed.

## Icons Folder
- [ ] Verify that all icons from the previous structure exist in the new structure.
- [ ] Ensure all icons load correctly in the application.

## Tests Folder
- [ ] Check that all existing test cases have been migrated.
- [ ] Verify that tests pass successfully in the new structure.
- [ ] Ensure that new tests (if added) are included and functioning.

## Docs Folder
- [ ] Confirm that all documentation files have been relocated accordingly.
- [ ] Verify links in documentation are correctly pointing to the new paths.

## Scripts Folder
- [ ] Ensure all scripts are present in the new scripts folder.
- [ ] Test the functionality of all scripts to confirm they work as intended.

## Cloud Folder
- [ ] Validate that cloud configurations are updated and correctly referenced.
- [ ] Ensure that deployment scripts are tested and working.

## Vendor Folder
- [ ] Verify that all vendor packages are included and correctly installed.
- [ ] Test any integrations that rely on vendor packages.

## Root PWA Files
- [ ] Ensure that the root files for the Progressive Web App (PWA) are present.
- [ ] Verify the PWA functions correctly in its new location.

## Final Steps
- [ ] Conduct a comprehensive review of the application to ensure everything is functioning properly.
- [ ] Document any issues or discrepancies found during verification.
