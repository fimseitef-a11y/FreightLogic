# FreightLogic PWA

## Purpose
The FreightLogic Progressive Web Application (PWA) is designed to streamline the freight management process, providing users with a responsive and efficient way to manage logistics and shipping operations.

## Quick Start Instructions
1. **Install Dependencies**: Ensure that you have the necessary tools installed, such as Node.js and npm.
   ```bash
   npm install
   ```
2. **Start the Application**: Run the following command to start the application locally:
   ```bash
   npm start
   ```
3. **Access the Application**: Open your web browser and go to `http://localhost:3000` to access the app.
